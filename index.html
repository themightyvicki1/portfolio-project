Weather Project

This purpose of this project was to design a weather app using an API for up-to-date information. It starts initially with a predetermined city. But the user is given the option to view the weather based on their current geographical location or by using the search bar to search for any city.
There is also the ability to view the weather for today plus five additional days. Along with this, the user is also able to view the humidity, wind speed, minimum temperature, and maximum temperature for the current day.
This project utilized a weather API, grids, arrays, event listeners, and various design techniques.

World clock Project

The purpose of this project was to design a simple world clock. It starts with the time and date that is updated every one second, for three predetermined cities. The user has the option to view the time and date for their current location plus three more predetermined cities from the dropdown box.
This project utilizes Moment.js, a dropdown box, functions, event listeners, and styling through CSS.

Landing Page Project

This project is a basic landing page that is meant to give the user information on baking so they can be informed. This project used a lot of CSS styling techniques such as borders, background colors, and different font colors.
It does include JavaScript where the user would be able to enter their email address and receive a recipe. It is only a front-end application so no information is stored, the purpose was the get familiar with JavaScript and CSS styling.

Transforming Images Project

This project included the introduction of divs and spans. Its purpose was to get familiar with them and learn how to transform images based on hovering over them. It also involved laying out the images with a grid and inserting them into each row and column.
It also includes a dark theme button that the user can press to change the styling of the page to darker colors.
